# 📊 Motorola Sales Dashboard (Power BI)

## 📌 Overview
This project is an interactive Sales Dashboard developed in Power BI...

## 🚀 Features
- Fully interactive dashboard with filters and slicers  
- Sales analysis by region, product, and time period  
- Key KPIs such as total sales, profit, and quantity sold  
- Trend analysis over time  

## 🛠 Tools & Technologies
- Power BI (Dashboard Development & Visualization)  
- Power Query (Data Cleaning & Transformation)  
- DAX (KPIs and calculated measures)  

## 📈 Key Insights
- Top-performing regions identified  
- Product-wise sales distribution analyzed  
- Monthly trends observed  
- Low-performance areas highlighted  

## 🎯 Objective
To design a dynamic and user-friendly dashboard for decision-making.

## 💡 What I Learned
- Power BI dashboard building  
- DAX measures  
- Data cleaning using Power Query  
- Visual storytelling
