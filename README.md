#📊 Motorola Sales Dashboard (Power BI)

##📌 Overview

This project is an interactive Sales Dashboard developed in Power BI to analyze and visualize Motorola’s sales data. It transforms raw data into meaningful insights through dynamic visuals, enabling better understanding of sales performance, trends, and key business metrics.

##🚀 Features
Fully interactive dashboard with filters and slicers
Sales analysis by region, product, and time period
Key KPIs such as total sales, profit, and quantity sold
Trend analysis to track performance over time
Clean and intuitive data visualizations for easy interpretation

##🛠️ Tools & Technologies
Power BI (Dashboard Development & Visualization)
Data Cleaning & Transformation (Power Query)
DAX (for calculated measures and KPIs)

##📈 Key Insights
Identified top-performing regions contributing to overall revenue
Analyzed product-wise sales distribution
Observed monthly/periodic trends to understand growth patterns
Highlighted areas with lower performance for potential improvement

##🎯 Objective
To design a dynamic and user-friendly dashboard that helps in data-driven decision-making by providing clear and actionable insights from sales data.

💡 What I Learned
Building interactive dashboards using Power BI
Writing DAX measures for business KPIs
Data cleaning and transformation using Power Query
Presenting data insights in a clear and visual format
